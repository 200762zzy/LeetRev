# LeetRev

本地 LeetCode 复习助手桌面应用。基于 Tauri v2 + React + Rust，使用本地 SQLite 存储。

## 功能

- **SM-2 间隔重复复习** — 回忆 → 对比 → 自评 流程，科学安排复习计划
- **内置代码编辑器** — 复习时直接写代码，支持 14 种语言语法高亮
- **全屏编码模式** — 一键全屏编辑器，专注解题
- **LeetCode 在线判题** — 提交代码到 LeetCode 判题；通过后强制进入自评阶段，必须 SM-2 评分后才能继续
- **多解法模块** — 每题可创建多个命名解法（标题 + 代码 + 复杂度 + 思路说明），支持排序、编辑、删除；复习对比阶段直接参考
- **草稿板** — 复习时内联展开右侧面板，左侧仅保留题目描述；支持文本笔记和画板手绘双模式，按题目自动保存
- **画板工具** — Canvas 画板支持画笔、颜色、粗细、橡皮擦、矩形/椭圆/直线、框选(select)移动像素、撤销/重做/清空
- **自定义 API 笔记** — 解题时添加/编辑/删除自定义 C++ STL API 条目，与内置数据合并
- **内联 API 速查** — 快速搜索 93+ 条 C++ STL API，包含签名、示例、复杂度、LeetCode 技巧
- **题目管理** — 增删改查 LeetCode 题目，支持标签、难度分类
- **LeetCode 同步** — 通过 LEETCODE_SESSION cookie 导入已解决/已尝试题目；同步时自动创建缺失标签（前缀和、线段树等 50+ 个），无需手动添加
- **复习历史** — 全局复习历史页面，展示所有题目的复习记录（SM-2 参数：ease factor、间隔天数、复习次数、下次复习时间），支持搜索排序和跳转题目
- **系统复习 / 专栏复习** — 全部待复习题目 / 按标签过滤的专题复习两种模式
- **代码模板** — 自动从 LeetCode 拉取方法签名
- **多版本代码** — 每题可保存多个语言版本的代码
- **统计看板** — 难度/标签分布、薄弱环节分析

## 快速开始

```sh
# 安装前端依赖
npm install

# 开发模式
npm run tauri dev

# 构建发布版本
npm run tauri build
```

## 使用指南

### 同步 LeetCode 进度

在设置页：
1. 点击「在浏览器中打开登录页」，登录 LeetCode
2. 打开 DevTools (F12) → Application → Cookies → 复制 LEETCODE_SESSION
3. 粘贴到设置页，点击「从 LeetCode 同步」
4. Cookie 会持久化保存，下次直接使用

### 复习模式

点击导航栏「复习」，进入复习页面。支持两种模式：

1. **系统复习** — 所有待复习题目按 SM-2 算法排列
2. **专栏复习** — 按标签分组，展示每个标签的待复习数量，点击进入专题复习

每次复习流程：

1. **回忆** — 查看题目描述，使用内置代码编辑器解答。可提交代码到 LeetCode 判题
2. **对比** — 对比自己的解答、历史代码、多解法参考。支持查看完整题目描述、思路笔记
3. **自评** — LeetCode 判题通过后强制进入此阶段，必须自评掌握度（容易/中等/困难）才能继续

提交 LeetCode 判题通过后，自动进入对比阶段并要求自评，确保 SM-2 复习周期完整记录。

### 草稿板

复习时点击代码编辑器工具栏的草稿板按钮（FileText 图标），右侧面板展开：

- **文本模式**：记录思路、伪代码、测试用例
- **画板模式**：切换 Canvas 画板，支持画笔、颜色、粗细、橡皮擦、矩形/椭圆/直线、框选(select)拖拽移动像素、撤销/重做/清空，适合手绘流程图和数据结构图
- 内容按题目自动持久化存储，文字和画板数据独立保存，切换模式不丢失

### 多解法

在题目详情页添加解法：

- 每道题可以创建多个命名解法（如"双指针"、"动态规划"、"哈希表"）
- 每个解法包含：标题、代码、语言、时间复杂度、空间复杂度、思路说明
- 支持 ↑↓ 排序、编辑、删除
- 复习对比阶段自动展示该题的所有解法作为参考
- 新建解法时，如果题目关联了 LeetCode，自动拉取对应语言的函数签名模板

### LeetCode 判题

复习回忆阶段（已配置 LeetCode cookie）：
1. 选择语言，编写代码
2. 点击「提交判题」
3. 等待实时判题结果：
   - 通过 / 答案错误 / 编译错误 / 运行错误
   - 失败时显示测试用例详情
4. 通过后自动进入对比+自评阶段；失败需修改后重新提交

### 复习历史

点击侧边栏「复习历史」，查看所有题目的复习记录：
- 统计卡片：总复习次数、复习题目数、平均 Ease Factor、平均间隔
- 完整 SM-2 参数：每次复习的 ease factor、间隔天数、复习次数、下次复习时间
- 搜索过滤：按题目名称搜索
- 排序切换：按复习时间、Ease Factor、间隔、下次复习时间排序
- 蓝色跳转按钮直接进入题目详情

### 自定义 API 条目

编写代码时，点击书签图标添加自定义 API 笔记：
- 添加方法级别的 C++ STL 条目，包含签名、示例、笔记
- 在 API 搜索面板中编辑或删除条目
- 自定义条目在搜索结果中带有特殊标记
- **草稿自动保存**：编辑时表单内容自动保存到本地，即使误关闭也不会丢失；API 面板的 `+` 按钮有琥珀色圆点提示未完成草稿

## 技术栈

- **前端**: React 19 + TypeScript + Vite 8 + TailwindCSS
- **后端**: Rust + Tauri v2
- **数据库**: SQLite (rusqlite)
- **间隔重复**: SM-2 算法
- **在线判题**: LeetCode.cn GraphQL API
